Phase 1: Setup Infrastructure

Konfigurasi PC lokal (Self-Hosted Runner) dan kebersihan repositori GitHub. (Dilakukan 1 kali di awal project).

1. Build PC Prep

  • Unreal Engine installed (matching project version).
  • Visual Studio with Game development with C++ workload.
  • Git for Windows with SSH Key configured (Auto-pull enabled).
  • GitHub Runner: Register in Settings > Actions > Runners > New self-hosted runner (Windows).

2. .gitignore Rules

Wajib diletakkan di root project untuk mencegah cache engine terupload.

Phase 2: Semantic Versioning

Standar penamaan versi dan penggunaan pre-release tags untuk identifikasi resiko *bug*.

Version Numbering Rules

FormatTypeWhen to Increment
v1.x.xMajorBreaking API changes, major engine version upgrades.
x.2.xMinorAdding new functionality that doesn't break existing builds.
x.x.3PatchSmall bug fixes, hotfixes, or optimizations.

Pre-Release Tags Strategy

-alpha

Experimental. Fitur baru, resiko bug sangat tinggi. Internal Devs.

-beta

Feature complete. Fokus ke pengujian stabilitas. Internal/External QA.

-rc

Release Candidate. Diyakini stabil, pending final sign-off Stakeholders.

Stable (None)

Production-ready. Siap diluncurkan ke Client / End User.

Phase 3 & 4: Automation Scripting

Integrasi Discord Webhook dan *blueprint* skrip GitHub Actions YAML untuk *runner*.

Discord Secret

Masuk ke pengaturan channel Discord > Integrations > Webhooks. Salin URL-nya dan masukkan ke GitHub Repository Settings > Secrets and variables > Actions.

Name: DISCORD_WEBHOOK

Script Verification

  • Pastikan variabel $UAT_Path mengarah tepat ke lokasi instalasi Unreal Engine di PC lokal.
  • Ubah `UE_5.3` pada skrip menjadi versi spesifik (misal 5.4) jika baru saja update engine.

.github/workflows/unreal_build.yml

name: Unreal Engine Build Automation
on:
  release:
    types: [published]

jobs:
  build_unreal:
    runs-on: self-hosted

    steps:
      - name: Checkout Code
        uses: actions/checkout@v4

      - name: Build & Cook (UAT)
        run: |
          $UAT_Path = "C:\Program Files\Epic Games\UE_5.3\Engine\Build\BatchFiles\RunUAT.bat"
          $ProjectFile = Get-ChildItem -Path "${{ github.workspace }}" -Filter *.uproject | Select-Object -First 1
          if (-not $ProjectFile) { throw "No .uproject file found!" }
          & $UAT_Path BuildCookRun -project="$($ProjectFile.FullName)" -noP4 -platform=Win64 -clientconfig=Development -cook -allmaps -build -stage -pak -archive -archivedirectory="${{ github.workspace }}\Build_Output"

      - name: Compress Build
        run: Compress-Archive -Path "${{ github.workspace }}\Build_Output\Windows\*" -DestinationPath "Build.zip" -Force

      - name: Upload to Release
        uses: svenstaro/upload-release-action@v2
        with:
          repo_token: ${{ secrets.GITHUB_TOKEN }}
          file: Build.zip
          asset_name: Build.zip
          tag: ${{ github.ref }}
          overwrite: true

      - name: Notify Discord
        if: always()
        uses: appleboy/discord-action@master
        with:
          webhook_id: ${{ secrets.DISCORD_WEBHOOK }}
          message: "**🚀 Build Complete!**\nStatus: ${{ job.status }}\nDownload: ${{ github.event.release.html_url }}"
